This page presents a template of the correct html structure for a tabbed interface using the tab control in PyEvolve's evo package. Note that the control has both a js and a css part, and that both must be referenced. Note also that the naming conventions used in this template must be observed: each tab has a unique prefix, and this string is attached to the td, tab div, and palette div ids of the html elements that make up that tab. (Prepared by Amanda Birmingham.)
